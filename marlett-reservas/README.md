# Marlett Reservations

Web-based reservation system for Marlett restaurant and event hall.

## Structure

```
marlett-reservas/
├── index.html          # Main reservation form
└── assets/             # Images and media files
    ├── marlett-fachada.jpg
    ├── marlett-lobby-1.jpg
    ├── marlett-rampa.jpg
    ├── marlett-lobby-2.jpg
    ├── marlett-auditorio.jpg
    ├── marlett-salon-montado.jpg
    └── marlett-arcos.jpg
```

## Setup

1. **Add your images** to the `assets/` folder
2. **Database**: Run the SQL from `../database.sql` in Supabase
3. **Deploy**: Push to Vercel or Netlify

## Features

- Beautiful dark theme UI
- Responsive design
- Supabase integration
- Form validation
- Preview mode
- Anti-bot protection

## Requirements

Images should be optimized for web:
- Format: JPG or WebP
- Size: < 500KB each
- Dimensions: ~1920x1080px (or similar aspect ratio)
